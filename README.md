# react-blog

## Overview


A blogging app with authentication and authorization using React, JWT, bCrypt and Express.

[https://steve-react-blog.herokuapp.com/](https://steve-react-blog.herokuapp.com/)


## Technology


* Node
* Express
* React
* React Router DOM
* PostgreSQL
* PG-Promise
* JWT
* Moment
* Body Parser
* Jest
* Enzyme
* React Shallow Renderer
